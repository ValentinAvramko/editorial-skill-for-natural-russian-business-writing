# Адаптер для Gemini

Статус: заготовка.

Используйте этот каталог для будущей упаковки редакторского стандарта под Gemini.

Первую реализацию лучше держать тонкой:

- добавить один основной файл инструкций;
- добавить локальные справочные файлы только если они реально улучшают поведение во время выполнения;
- оставить слой оценки общим в `../../core/`.

Перед переводом адаптера в `active`:

1. определить нативный для платформы файл инструкций или набор таких файлов;
2. перенести в этот формат логику из `../../core/prompt-spec.md`;
3. решить, нужны ли локальные копии `patterns` и `examples`;
4. проверить поведение на `../../core/eval-cases.md`.
