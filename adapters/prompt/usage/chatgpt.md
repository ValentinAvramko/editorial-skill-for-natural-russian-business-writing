# Использование prompt в ChatGPT

`PROMPT.md` рассчитан на ручную вставку в обычный чат без дополнительных файлов.
`references/` используются как дополнительные текстовые материалы: ChatGPT не откроет их по имени файла, поэтому нужный файл или его релевантный фрагмент надо вставлять в диалог вручную.

## Базовый сценарий

1. Открой новый диалог.
2. Скопируй `PROMPT.md` целиком.
3. Сразу под ним добавь короткий контекст и исходный текст.

Пример:

```text
Это сопроводительное письмо.
Если текст уже нормальный, правь минимально.
Нужен только итоговый вариант.

Текст:
...
```

## Если нужен более осторожный режим

Добавь перед текстом:

```text
Сохрани мой тон и формулировки там, где они уже звучат нормально.
Правь только явные AI-штампы, канцелярит и слишком гладкие места.
```

## Как использовать references

Подключай `references` не вместо `PROMPT.md`, а как дополнительный контекст после основного prompt-а.

- Используй [`adapters/prompt/references/patterns.md`](../references/patterns.md), когда текст явно отдаёт нейроязом, канцеляритом, карьерным шаблоном или корпоративным жаргоном.
- Используй [`adapters/prompt/references/examples.md`](../references/examples.md), когда нужно точнее удержать глубину правки и не переисправить уже нормальный текст.
- Если задача узкая, лучше вставлять не весь reference-файл, а нужный фрагмент.
- Если текст сложный или пограничный, можно вставить файл целиком.

Практический порядок такой:

1. Сначала вставь `PROMPT.md`.
2. Ниже добавь короткую рамку задачи.
3. Затем вставь релевантный reference-фрагмент или целый reference-файл.
4. После этого вставь исходный текст.

Пример с `patterns.md`:

```text
Ниже даю ориентир по паттернам, которые особенно важно убирать в этом тексте.
Используй его как дополнительный контекст, но не копируй формулировки механически.

[вставка нужного фрагмента из patterns.md]

Текст:
...
```

Пример с `examples.md`:

```text
Ниже даю несколько before/after-примеров как ориентир по степени вмешательства.
Смотри на тон и глубину правки, но не копируй сами формулировки.

[вставка нужного фрагмента из examples.md]

Текст:
...
```

## Если нужен второй проход

После первого ответа можно дать короткое уточнение:

```text
Сделай еще один проход.
Убери остаточную гладкость и карьерные штампы, но не упрощай текст до сухого телеграфного стиля.
Верни только итоговый вариант.
```

## Что считать хорошим результатом

- смысл и факты не изменились;
- текст стал естественнее, а не просто глаже;
- не появились новые достижения, мотивация или эмоции;
- ответ не превратился в объяснение вместо редактуры.
