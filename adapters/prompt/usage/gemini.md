# Использование prompt в Gemini

1. Открой `PROMPT.md`.
2. Скопируй его целиком в начало диалога.
3. Добавь исходный текст.
4. При необходимости уточни жанр, глубину правки и нужен ли комментарий к изменениям.

Базовый запрос:

```text
Перепиши этот текст так, чтобы он звучал естественно и по-деловому.
Не придумывай новые факты и не делай его более рекламным.
Нужен только итоговый вариант.

Текст:
...
```

Если текст слишком гладкий, канцелярский или карьерно-шаблонный, при необходимости дополнительно подключай `references/patterns.md`.
Если нужен ориентир по степени вмешательства, используй `references/examples.md`.
